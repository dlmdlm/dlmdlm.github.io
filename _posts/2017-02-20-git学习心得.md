# git操作过程

## 把本地文件推送到远程仓库

 1. 进到的仓库（你要进行修改的）

 > cd dlmdlm.github.io

 2. 添加文件到缓存区

 > git add "路径/文件名"

 3. 把缓存区的文件添加到仓库

 > git commit -m "注释"
 
 4. 把本地的仓库的文件推到远程仓库

> git push origin master
